# Interface definition

## Interface definition of the whole machine

ROC-RK3568-PC provides rich interfaces, including:

* Power interface
* 1 x USB3.0(host/device)
* 1 x USB3.0（host)
* 4 x USB2.0 (interface 2, seat 2)
* HDMI
* MIPI screen interface
* EDP screen interface
* TP touch interface
* Ethernet
* WIFI antenna
* bluetooth antenna
* MIC interface
* Linein interface
* 3.5 mm headphone jack
* 12 v power supply interface
* IR interface
* TF card slot
* SIM card slot
* SATA 3.0 interface
* Speaker interface
* Power button
* Reset button
* Recovery button
* Debugging serial port
* Industrial serial port (RS485,RS232,TTL)
* MIPI camera interface
* MINI-PCIE interface (options 5G module M.2 NGFF interface)
* M.2 PCIE3.0 interface
* 2 x CAN/CANFD
