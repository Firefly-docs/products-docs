# Interface Definition

## Machine interface definition

ROC-RK3566-PC provides a wealth of interfaces, mainly including:

* Power interface
* 1 x USB3.0 (host)
* 4 x USB2.0 (seat)
* 1 x HDMI
* 2 x MIPI screen interface
* 1 x EDP screen interface
* 1 x 1G Ethernet
* 1 x 2.4/5G wifi
* 1 x WIFI antenna
* 1 x bluetooth
* 1 x MIC interface
* 1 x 3.5mm Phone Jack
* 1 x DSpeaker interface
* 1 x Type-C(host/device)
* 1 x 12v power supply interface
* 1 x IR interface
* 1 x TF card slot
* 1 x SIM slot
* 1 x Power key
* 1 x Reset key
* 1 x Recovery key
* 1 x Debug serial
* 1 x Industrial serial port (RS485,RS232,TTL)
* 1 x MINI-PCIE interface (options 5G module M.2 NGFF interface)
* 1 x M.2 PCIE2.0 interface
* 1 x SATA interface(M.2 PCI interface multiplexing)
* 1 x Nor flash
* 1 x MIPI camera interface
