# Interface definition

## Machine interface definition

AIO-3399ProC provides a wealth of interfaces, mainly including:

* Power interface
* Type-C (OTG)
* 1 x USB3.0
* 2 x USB2.0
* HDMI
* Ethernet
* LVDS screen interface
* EDP screen interface
* TP touch interface
* Screen voltage jumper interface
* Backlight interface
* WIFI antenna
* Bluetooth antenna
* Power button
* MIC interface
* Audio input and output (linein, lineout)
* 3.5mm headphone jack
* RTC power connector
* 12V power interface
* IR interface
* TF card slot
* SIM card slot
* Extended button interface
* I2C
* I2S
* Speaker interface
* recovery button
* EDP screen interface
* Debug serial port
* Industrial-grade serial port (RS485, RS232, 2TTL)
* CAN bus
* RTC power inlet
* MIPI screen interface (dual LVDS multiplexing)
* MINI-PCIE

      - WiFi router
