# Interface definition

## Firefly-RK3399 Interface Definition

Firefly-RK3399 provides a wealth of interfaces, including:

* Power interface,
* USB3.0 HOST,
* Digital optical fiber,
* Type-C,
* HDMI,
* Ethernet,
* WIFI antenna,
* Bluetooth antenna,
* Dual MIPI camera,
* Fan interface,
* Reset button,
* Power button,
* Silicon wheat,
* Audio input and output,
* RTC power interface,
* Infrared reception,
* TF card slot,
* USB2.0 HOST,
* Recovery key,
* EDP screen interface,
* DVP camera interface,
* Debug serial port,
* PCIE interface,
* Battery power input port,
* MIPI screen interface,
* MINI-PCIE.

## Shipping list(For reference only)

* Firefly-RK3399 Board x1
* Type-C data cable x1
* Copper tube antenna  ×1

Additionally, the following accessories or devices are needed:
*   12V-2A Power adapter
*   Display
      - Display or TV which supports VGA interface, and VGA cable
      - Display or TV which supports HDMI interface, and HDMI cable
*   Network
      - 100M/1000M ethernet cable, and corresponding router
      - WiFi router
*   Input devices
      - USB wire/wireless mouse/keyboard
      - Infrared remote controly
