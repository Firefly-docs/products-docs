# Introduction
AIO-Orin NX is equipped with the official NVIDIA Jetson Orin NX core board module, featuring 16GB of memory and a computing power of up to 100 TOPS. It can run all modern AI models, including Transformer and ROS robot models. It enables larger and more complex deep neural networks, utilizing TENSORFLOW, OPENCV, JETPACK, KERASMXNET, PY-TORCH, etc., to achieve functionalities such as object recognition, target detection and tracking, speech recognition, and other visual development, meeting the demands of higher AI application scenarios.
