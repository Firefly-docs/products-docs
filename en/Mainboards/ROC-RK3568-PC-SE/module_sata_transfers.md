# [SATA Adapter](https://www.firefly.store/products/sata-adapter)
ROC-RK3568-PC-SE needs  SATA Adapter module when using the SATA storage device. The SATA Adapter fits all series mainboards of Firefly with FPC SATA interface, can be connected to 2.5'' or 3.5'' SSD/HDD, and it uses FPC ribbon cable, with aluminum foil shielding, minimizes signal interference
