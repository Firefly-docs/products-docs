# <font color="red">NPU use</font>

The NPU module is built into the 3399pro chip of Station-P1-Pro.

##Simple usage example
Connect the camera, open Settings->Apps & notification->RK3399Pro Pose Demo->Permission, give Camera and storage permissions, and then open the desktop application RK3399Pro Pose Demo to run.
