# Interface definition

## Machine interface definition

Station-P1-Pro provides a wealth of interfaces, mainly including:


* Power interface
* 1 x USB3.0 (host/device)
* 8 x USB2.0 (interface×6, socket×2)
* HDMI, Ethernet
* LVDS screen interface
* eDP screen interface
* TP touch interface
* Screen voltage jumper interface
* Backlight interface
* WIFI antenna
* Bluetooth antenna
* Power button
* MIC interface
* Audio input and output (linein, lineout)
* 3.5mm headphone jack
* RTC power interface
* 12v power interface
* IR interface
* TF card slot
* SIM card slot
* Extended button interface
* Speaker interface
* USB2.0 HOST
* recovery button
* Debug serial port
* Industrial-grade serial port (RS485, RS232, 2TTL)
* MIPI screen interface (double lvds multiplexing)
* MINI-PCIE

The details are as follows:

![](../../../rk3399_img/Station-P1-Pro/interface.png)

In addition to the above interfaces, AIO-3399JD4 (with SPR5801S chip attached) supports NPU acceleration.

**Note: The SPR5801S chip in AIO-3399JD4 is optional. If you need the NPU acceleration function, please pay attention to purchasing the hardware version with the SPR5801S chip. (The hardware NPU has been replaced with SPR5801S, but the software still supports the old version SPR2801S)**
