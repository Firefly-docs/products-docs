
ROC-RK3399-PC-PLUS is a mini PC motherboard, about the size of a small phone.
![](../../../rk3399_img/ROC-RK3399-PC-PLUS/pc+_interface_front.jpg)

![](../../../rk3399_img/ROC-RK3399-PC-PLUS/pc+_interface_back.jpg)

* Dual-core Cortex-A72 up to 1.8GHz & Quad-core Cortex-A53 up to 1.5GHz Six Core Processor
* Mali-T864 GPU，Support OpenGL ES1.1/2.0/3.0, OpenCL1.2, DirectX11.1.
* LPDDR4 RAM (4GB)
* 3 x USB 2.0
* 10/100/1000Mb Eth interface
* IR Infrared Module
* HDMI 2.0a with HDCP 2.2
* DisplayPort 1.2 (4K 60Hz)
* 3 x MIPI interfaces :MIPI0 only for DSI, MIPI1 for DSI or CSI, MIPI2 only for CSI
* eDP 1.3
* I2S，Support 8 Channels
* High speed eMMC Expansion Interface
* MicroSD (TF) Card Slot
* 1 x Type-C Interfaces :Type-C0 support DP 1.2、USB3.0
* RTC : battery model -- CR1220 ; Battery seat -- CR1220,BS-12-A3B0J002
* 132 Pin Expansion  Interfaces:I2C、SPI、Uart、I2S....
