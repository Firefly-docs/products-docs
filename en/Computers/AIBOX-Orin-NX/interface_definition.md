# Interface definition

AIBOX-Orin NX has rich interfaces, mainly including:
- 12V DC Power（5.5*2.5mm）
- Power Key
- Recovery Key (Put AIBOX-Orin-NX into Force Recovery Mode, to flash images)
- 1000Mbps Ethernet x 1
- USB 3.0 x 2
- HDMI
- NVME M.2(PCIE3.0 x 1)
- Console (Debug)
- Type-C（USB2.0 OTG）
- Power Led
