# Interface definition

## Firefly-RK3399 Interface Definition

Firefly-RK3399 provides a wealth of interfaces, including:

* Power interface,
* USB3.0 HOST,
* Digital optical fiber,
* Type-C,
* HDMI,
* Ethernet,
* WIFI antenna,
* Bluetooth antenna,
* Dual MIPI camera,
* Fan interface,
* Reset button,
* Power button,
* Silicon wheat,
* Audio input and output,
* RTC power interface,
* Infrared reception,
* TF card slot,
* USB2.0 HOST,
* Recovery key,
* EDP screen interface,
* DVP camera interface,
* Debug serial port,
* PCIE interface,
* Battery power input port,
* MIPI screen interface,
* MINI-PCIE.
