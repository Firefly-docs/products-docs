# NETWORKS 
## Introduction 
### Cellular (4G/5G)
the default network card name is `wwan0`.
### Wireless Network (WIFI/BT)
the default network card name is `wlP1p1s0`.
### Gigabit Ethernet
the default network card name is `eth2` and `enP5p1s0`.
### Multi-Gigabit Ethernet
the default network card name is `eno1`.
