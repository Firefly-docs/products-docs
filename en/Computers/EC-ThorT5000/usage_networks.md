# NETWORKS 
## Introduction 

### Cellular (4G/5G)
the default network card name is `wwan0`.
### Wireless Network (WIFI/BT)
the default network card name is `wlP1p1s0`.
### Gigabit Ethernet
the default network card name is `enP2p1s0`.
### Multi-Gigabit Ethernet
the default network card name is `mgbe0`, `mgbe1`,`mgbe2`,`mgbe3`.
