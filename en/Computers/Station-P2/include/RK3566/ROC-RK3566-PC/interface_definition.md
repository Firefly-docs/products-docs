# Interface Definition

## Machine interface definition

Station-P2 provides a wealth of interfaces, mainly including:

* 1 x usb3.0 (host)
* 1 x usb3.0 (host)
* 1 x hdmi
* 1 x mipi dsi
* 1 x 1G eth interface
* 1 x 2.4/5G wifi
* 1 x 3.5mm Phone Jack
* 1 x Type-C(PD) power interface
* 1 x ir interface
* 1 x tf card slot
* 1 x power key
* 1 x reset key
* 1 x recovery key
* 1 x debug serial
* 1 x Nor flash
* 1 x mipi csi
