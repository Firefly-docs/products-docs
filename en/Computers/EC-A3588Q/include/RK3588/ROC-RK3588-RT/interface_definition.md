# Interface definition

## Interface definition of the whole machine

**EC-A3588Q V1.0** provides rich interfaces, including:

* 2 x HDMI2.1 (Max output support 8K@60Hz)
* 1 x Display Port1.4 / Type-C OTG (Max output support 8K@30Hz)
* 1 x PCIe2.0 (M.2 NVMe)
* 1 x PCIe2.0 (M.2 E-KEY for WiFi6/BT5.0)
* 1 x TF Card
* 1 x USB3.0
* 1 x USB2.0
* 2 x RJ45（Suport 1Gbps）
* 1 x RJ45 (Suport 2.5Gbps)
* 1 × 3.5mm Headphone
* 1 x BTB Connector
