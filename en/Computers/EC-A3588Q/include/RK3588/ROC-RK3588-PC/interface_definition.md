# Interface Definition

## Whole machine interface definition

**EC-A3588Q** provides a rich set of interfaces, mainly including.

* 1 x HDMI2.1 (supports up to 8K@60Hz output)
* 1 x HDMI2.0
* 1 x HDMIIN2.0 (supports up to 4K@60Hz input)
* 1 x Display Port1.4 (up to 8K@30Hz output)
* 1 x USB3.0 OTG (Type-C)
* 1 x PCIe3.0x4
* 1 x TF Card
* 2 x USB3.0
* 1 x WIFI (wifi6 support)
* 1 x Bluetooth
* 1 x FAN
* 1 x RJ45 (supports 1Gbps)
* 1 x HeadPhone
* 1 x 12V Power Input
* 1 x RTC
* 3 x KEY
* On-board Expansion Pin Interface
    * 1 x Debug
    * 1 x SPI
    * 1 x I2C
    * 1 x LINEOUT
    * 1 x UART
    * 2 x ADC
    * 1 x CAN (RX + TX)
* Expansion board 164 Pin interface
    * 5G/4G LTE
    * GMAC
    * MIPI-CSl
    * MIPI-DSI
    * VGA
    * DP1.4 
    * USB3.0
    * I2C
    * PCle2.0
    * UART
    * CAN
    * I2C
    * SPDIF
