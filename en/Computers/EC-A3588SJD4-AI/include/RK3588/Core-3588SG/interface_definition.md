# Interface definition

## Interface definition of the whole machine

AIO-3588SJD4-AI provides rich interfaces, including:

* 1 x Display Port1.4 (Max output support 8K@30Hz)
* 1 x USB3.0 OTG(Type-C)
* 2 x MIPI-DSI
* 1 x EDP
* 1 x MIPI-CSI
* 1 x MIPI-D/CPHY-CSI
* 1 x TF Card
* 1 x USB3.0
* 1 x USB2.0(One USB2.0 are led by a 40-pin pin row)
* 1 x WIFI (Support wifi6)
* 1 x Bluetooth
* 1 x FAN
* 2 x RJ45 (Suport 1Gbps)
* 1 x Speaker
