# Interface definition

## Interface definition of the whole machine

**EC-R3588SPC V1.0** provides rich interfaces, including：

* 1 x HDMI2.1 (Max output support 8K@60Hz)
* 1 x HDMI2.0
* 1 x HDMIIN2.0 (Max input support 4K@60Hz)
* 1 x Display Port1.4 (Max output support 8K@30Hz)
* 1 x USB3.0 OTG(Type-C)
* 2 x MIPI-DSI
* 2 x MIPI-CSI
* 1 x M.2 SATA3.0 或 M.2 PCIe2.0
* 1 x PCIe3.0x4
* 1 x TF Card
* 1 x 4G LTE
* 1 x SIM 卡槽
* 2 x USB3.0
* 2 x USB2.0 (Lead out from 44Pin)
* 1 x PCIe2.0 (M.2 E-KEY for WiFi6/BT5.0)
* 1 x FAN
* 1 x RJ45 (Suport 1Gbps)
* 1 × 3.5mm headphone jack
