# Interface Introduction

The EC-Orin NX interface is rich and mainly includes:
- 12V power interface (5.5*2.5mm)
- POWER button
- RS232
- RS485
- CAN
- Gigabit Ethernet x 1
- USB 3.0 x 2
- HDMI
- SIM card slot
- BT antenna
- WIFI antennas x 2
- 4G antenna
- Headphone
- NVME interface (PCIE3.0 x 1)
- Type-C (USB2.0, but defaults to debugging serial port)
- Power indicator light
