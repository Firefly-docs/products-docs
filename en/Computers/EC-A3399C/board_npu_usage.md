# NPU use

The AIO-3399JD4 core board has a built-in GTI 2801S/5801S NPU module.

**Note: To use the NPU function, please purchase the AIO-3399JD4 hardware version with SPR2801S/SPR5801S chip attached**

## Instructions

For how to use the AI module, please refer to [GTI NPU](http://wiki.t-firefly.com/en/gti-npu/) related documents.
