# Product introduction

Station P2 Embedded Computer, Based on AIO-3566JD4 high-performance 
open source platform . Moreover, it supports 4K hardware acceleration, dual-screen identical display/dual-screen differential display and 
has rich interface for easy connection of various industrial equipment. 
It is equipped with ARM Cortex-A55 architecture, quad-core 64-bit high-performance processor, 
frequency up to 2.0GHz, integrates GPU, support RKNN NPU AI,supports H.265 HEVC and VP9, H.264 
encoding, 4K HDR, and has the maximum support of 4K hardware decoding.
