# Interface definition

## Interface definition of the whole machine

AIO-3566JD4 provides rich interfaces, including:

* Power interface
* 1 x Type C(host/device)
* 1 x USB3.0（host)
* 2 x USB2.0（host)
* HDMI
* MIPI screen interface
* EDP screen interface
* Ethernet
* WIFI antenna
* bluetooth antenna
* 3.5 mm headphone jack
* 12 v power supply interface
* TF card slot
* SATA 3.0 interface
* Speaker interface
* Power button
* Reset button
* Recovery button
* MaskRom button
* Debugging serial port
* Industrial serial port (RS485,RS232,TTL)
* MIPI camera interface
* M.2 PCIE3.0 interface
