# Wireless module


Firefly supports the following Wireless modules:

- [EC20 4G-LTE module](https://wiki.t-firefly.com/en/EC20/ec20.html)
- [EG25-G 4G LTE modue](https://wiki.t-firefly.com/en/EG25/eg25.html)
- [RM500Q-GL 5G module](https://wiki.t-firefly.com/en/RM500Q-GL-5G/rm500q-gl.html)
- [GPS & GLONASS & RG4538](https://wiki.t-firefly.com/en/GNSS/gnss.html)
- [WCT1FR2201 WiFi/BT module](https://wiki.t-firefly.com/en/RTL8822CE/wct1fr2201.html)
- [WXT14R2201 WiFi/BT module](https://wiki.t-firefly.com/en/RTL8852BE/wxt14r2201.html)
