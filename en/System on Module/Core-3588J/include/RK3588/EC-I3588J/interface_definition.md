# Interface definition

## Interface definition of the whole machine

**ITX-3588J  V1.3** provides rich interfaces, including:

* 1 x HDMI2.1 (Max output support 8K@60Hz)
* 1 x HDMI2.0
* 1 x HDMIIN2.0 (Max input support 4K@60Hz)
* 1 x Display Port1.4 (Max output support 8K@30Hz)
* 1 x USB3.0 OTG(Type-C)
* 1 x VGA
* 2 x MIPI-DSI
* 1 x MIPI-CSI
* 4 x SATA3.0
* 1 x M.2(SATA3.0)
* 1 x PCIe3.0x4
* 1 x TF Card
* 1 x 4G LTE
* 1 x 5G Mobile Network
* 1 x SIM 卡槽
* 4 x USB3.0
* 4 x USB2.0
* 1 x WIFI (Support wifi6）)
* 1 x Bluetooth
* 1 x FAN (<font color=#FF0000>V1.1 version and subsequent versions is 4 PIN, controlled by PWM, ADC detection access</font>)
* 2 x RJ45 (Suport 1Gbps)
* 1 x Speaker
* 1 x Line-In
* 1 x RS485
* 1 x RS232
* 1 x CAN (<font color=#FF0000>V1.1 version added, H and L markings reversed, and screen printing correct for V1.2 and subsequent versions</font>)
* Multiple power input interfaces
