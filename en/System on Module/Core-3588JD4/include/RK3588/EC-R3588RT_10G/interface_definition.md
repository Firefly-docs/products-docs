# Interface definition

## Interface definition of the whole machine

Core-3588JD4 provides rich interfaces, including:

* 1 x HDMI2.1 (Max output support 8K@60Hz)
* 1 x USB3.0 OTG(Type-C)
* 1 x Display Port1.4 (Max output support 8K@30Hz)
* 1 x 4G LTE
* 1 x SIM card slot
* 1 x CAN
* 3 x RS232
* 2 x RS485
* 1 x SATA3.0 or PCIe2.0
* 1 x RELAY（Relay output）
* 1 x INPUT（Optocoupler isolation input）
* 3 x LED（One tricolor led, two monochrome leds）
* 1 x TF Card
* 1 x USB3.0
* 1 x USB2.0 (Two USB2.0 are led by a 20-pin pin row)
* 1 x WIFI
* 1 x Bluetooth
* 2 x RJ45 (One supports 1Gbps, and the other supports 100Mbps)
* 1 × 3.5mm Headphone (<font color=#FF0000>V1.1 version support recording</font>)
