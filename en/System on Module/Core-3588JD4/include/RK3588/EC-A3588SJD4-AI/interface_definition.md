# Interfaces Definition

AIO-3588JD4 provides many interfaces, includes:

* 1 x HDMI2.1 (Up to 8K@60Hz)
* 1 x USB Type-C (For download/debug)
* 2 x USB3.0
* 2 x MIPI-DPHY-CSI
* 1 x MIPI-D/CPHY-CSI
* 1 x TF Card Slot
* 1 x 4G LTE
* 1 x SIM Card Slot
* 1 x USB2.0 (Through pin header)
* 1 x WIFI (External module)
* 1 x Bluetooth (External module)
* 1 x FAN
* 1 x RJ45 (Support 1Gbps ethernet)
* 1 x Line-Out
* 1 x Line-In
* 1 x Headphone
* 1 x CAN
* 1 x RS232
* 1 x RS485
* 1 x SPI
* 2 x I2C
