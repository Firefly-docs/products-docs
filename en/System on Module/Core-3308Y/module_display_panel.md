# screen mod

## 4.0 inch RGB LCD module

### Product Parameters

* Model: HC480480TFT40-499-CT
* Size: 4.0 inches
* Resolution: 480x480
* Display interface: RGB
* Panel material: IPS panel
* Touch screen: multi-point capacitive touch

<!-- ### Reference firmware -->

### compile

When compiling the supported 4.0-inch screen firmware with the official website SDK, pay attention to the following:
</br>

* Board-level configuration file for ROC-RK3308B-CC-PLUS: `rk3308b-roc-cc-plus-amic-rgb_4.0inch_emmc.dts`
* Configuration file of screen module: `rk3308b-rgb_4.0inch_ST7701S.dtsi`
