# Interface definition

## Interface definition of the whole machine

AIO-3562JQ provides rich interfaces, including:

* Power supply
* 1 x USB3.0(host/otg)
* 3 x USB2.0 (typeA x 1, 4-pin seat x 2)
* MIPI-DSI screen interface
* 1000M Ethernet
* 100M Ethernet
* WIFI/BT antenna
* MIC interface
* Speaker interface
* TF card slot
* SIM card slot
* Power button
* Reset button
* Recovery button
* Maskrom button
* Debugging serial port
* Industrial serial port (RS485,RS232,UART)
* Industrial I/O (input x 2, output x 2)
* CAN/CANFD
* 2 x MIPI camera interface
* MINI-PCIE interface (USB2.0)
* I2C
* Battery interface
