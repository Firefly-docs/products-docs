# Interface definition

## Interface definition of the whole machine

AIO-3588JQ  provides rich interfaces, including:

* 1 x HDMI2.1 (Max output support 8K@60Hz)
* 1 x Display Port1.4 (Max output support 8K@30Hz)
* 1 x USB3.0 OTG(Type-C)
* 2 x MIPI-DSI
* 1 x MIPI-CSI
* 2 x MIPI-D/CPHY-CSI
* 1 x SATA2.0
* 1 x M.2(SATA2.0)
* 1 x TF Card
* 1 x 4G LTE
* 1 x 5G Mobile Network
* 1 x SIM Card Slot
* 1 x USB3.0
* 4 x USB2.0(Two USB2.0 are led by a 9-pin pin row)
* 1 x WIFI (Support wifi6)
* 1 x Bluetooth
* 1 x FAN
* 2 x RJ45 (Suport 1Gbps)
* 1 x Speaker
* 1 x Line-In
* 1 x MIC
* 1 x POE
