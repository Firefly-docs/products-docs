# Interface definition

## Interface definition of the whole machine

AIO-3568J provides rich interfaces, including:

* 12V power supply interface
* 4 x USB3.0 (otg×1, host×3)
* 4 x USB2.0 (interface 1, seat 3)
* HDMI
* VGA
* MIPI screen interface
* LVDS screen interface
* Ethernet
* MIC interface
* 3.5mm headphone jack
* TF card slot
* SIM card slot
* SATA 3.0 interface
* FAN interface
* Speaker interface
* Power button (seat)
* Reset button
* Recovery button
* Debug serial port
* Industrial serial port (RS485, RS232, TTL)
* Expansion port (ADC, I2C, GPIO)
* MIPI camera interface
* MINI-PCIE 4G interface
* M.2 5G interface
* PCIe 3.0 interface
* M.2 SATA 3.0 interface
* CAN/CANFD
