# 接口定义

## 整机接口定义

AIO-186JD4 提供了丰富的接口，主要包括：

* 12V 电源接口
* 2 x USB 3.0
* 2 x 千兆以太网
* WIFI 天线
* 风扇接口
* Power 按键
* Reset 按键
* Recovery 按键
* 调试串口(TTL)
* CAN 
* 工业级串口 (RS485, RS232 )
* Type-C (USB2.0,调试串口) 
* SIM 卡槽
* Mini PCIe 接口(4G LTE)
* M.2 接口(5G Module)
* NVME 接口(PCIE3.0 x 1)
* SATA 3.0
* TF 卡槽
* 耳机
* RTC 
* MIPI CSI 
* MIPI DSI
* HDMI 2.0
