# 接口定义

## 整机接口定义

**Core-3588L** 提供了丰富的接口，主要包括：

* 1 x HDMI2.1（最大支持 8K@60Hz 输出）
* 1 x HDMI2.0
* 1 x HDMIIN2.0（最大支持 4K@60Hz 输入）
* 1 x Display Port1.4（最大支持 8K@30Hz 输出）
* 1 x USB3.0 OTG(Type-C)
* 1 x PCIe3.0x4
* 1 x TF Card
* 2 x USB3.0
* 1 x WIFI（支持 wifi6）
* 1 x Bluetooth
* 1 x FAN
* 1 x RJ45（支持 1Gbps）
* 1 x HeadPhone
* 1 x 12V Power Input
* 1 x RTC
* 3 x KEY
* 板载扩展 Pin 接口
    * 1 x Debug
    * 1 x SPI
    * 1 x I2C
    * 1 x LINEOUT
    * 1 x UART
    * 2 x ADC
    * 1 x CAN（RX + TX）
* 扩展板 164 Pin 接口
    * 5G/4G LTE
    * GMAC
    * MIPI-CSl
    * MIPI-DSI
    * VGA
    * DP1.4 
    * USB3.0
    * I2C
    * PCle2.0
    * UART
    * CAN
    * I2C
    * SPDIF
