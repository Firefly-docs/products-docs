
# 接口定义

## 整机接口定义

Core-3566JD4 提供了丰富的接口，主要包括：

* 电源接口
* 1 x USB3.0 (host接口)
* 4 x USB2.0 (座子)
* HDMI
* 2 x MIPI 屏幕接口
* EDP 屏幕接口
* TP 触摸接口
* 以太网
* WIFI 天线
* 3.5mm 耳机接口
* MIC 接口
* 喇叭接口
* 12V 电源接口
* Type-C(host/device)
* IR 接口
* TF 卡槽
* SIM 卡槽
* Power 按键
* Reset 按键
* Recovery 按键
* 调试串口
* 工业级串口 (RS485, RS232, TTL)
* NOR flash
* MIPI 摄像头接口
* M.2 PCIE 2.0接口
* SATA 接口
