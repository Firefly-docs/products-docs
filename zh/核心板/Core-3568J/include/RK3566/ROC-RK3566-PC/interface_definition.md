# 接口定义

## 整机接口定义

Core-3568J 提供了丰富的接口，主要包括：

* 1 x USB3.0 (host)
* 1 x USB2.0 (host)
* HDMI
* MIPI 屏幕接口
* TP 触摸接口
* 以太网
* WIFI 天线
* 3.5mm 耳机接口
* Type-C(PD)电源接口
* IR 接口
* TF 卡槽
* Power 按键
* Reset 按键
* Recovery 按键
* 调试串口
* NOR flash
* MIPI 摄像头接口
