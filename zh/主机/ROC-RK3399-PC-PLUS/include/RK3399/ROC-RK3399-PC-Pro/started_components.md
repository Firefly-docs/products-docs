


ROC-RK3399-PC-PLUS 是一款迷你PC主板,体积只有小型手机的大小。
![](../../../rk3399_img/ROC-RK3399-PC-PLUS/pc+_interface_front.jpg)

![](../../../rk3399_img/ROC-RK3399-PC-PLUS/pc+_interface_back.jpg)

* Dual-core Cortex-A72 up to 1.8GHz & Quad-core Cortex-A53 up to 1.5GHz 六核处理器
* Mali-T864 GPU，支持 OpenGL ES1.1/2.0/3.0, OpenCL1.2, DirectX11.1.
* LPDDR4 RAM (4GB)
* 3 x USB 2.0
* 10/100/1000Mb 网口
* IR红外模组
* HDMI 2.0a with HDCP 2.2
* DisplayPort 1.2 (4K 60Hz)
* 3 x MIPI 接口:MIPI0 only for DSI, MIPI1 for DSI or CSI, MIPI2 only for CSI
* eDP 1.3
* I2S，支持 8 通道
* 高速 eMMC 扩展接口
* MicroSD (TF) 卡槽
* 1 x Type-C 接口:Type-C0 支持DP 1.2、USB3.0
* RTC : 电池型号 -- CR1220 ; 座子 -- CR1220,BS-12-A3B0J002（2PIN电池座卧式灰色镀金1U）锦和泰
* 132 个扩展接口:I2C、SPI、Uart、I2S....
## 发货清单（仅供参考）

## 配件
在使用过程中，你可能需要以下配件：
*   12V-2A 电源适配器
* 	Firefly 串口模块
*   显示设备
    * 	带 HDMI 接口的显示器或电视，及 HDMI 连接线
*   网络
    *   100M/1000M 以太网线缆，及有线路由器
    *   WiFi 路由器
*   输入设备
    *   USB 无线/有线的鼠标/键盘
    *   红外遥控器
