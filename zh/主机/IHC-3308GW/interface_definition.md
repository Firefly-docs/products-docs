
# 接口定义

## 整机接口定义

IHC-3308GW 提供了丰富的接口，主要包括：

* 电源接口
* 1 x Type-C
* 1 x USB3.0
* 2 x USB2.0
* HDMI
* MIPI 屏幕接口
* EDP 屏接口
* 以太网
* WIFI 天线
* 蓝牙天线
* 3.5mm 耳机接口
* 12V 电源接口
* TF 卡槽
* SATA 接口
* Power 按键
* Reset 按键
* Recovery 按键
* MaskRom 按键
* 调试串口
* 工业级串口 (RS485, RS232, TTL)
* MIPI 摄像头接口
* M.2 PCIE 3.0接口
