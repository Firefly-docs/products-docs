# 接口介绍

EC-A186JD4 接口丰富，主要包括：
- 12V 电源接口（5.5*2.5mm）
- POWER 按键
- RS232
- RS485
- CAN
- 千兆以太网 x 2
- USB 3.0 x 2
- HDMI
- TF 卡座
- SIM 卡座
- BT 天线
- WIFI 天线 x 2
- 4G 天线
- 耳机
- NVME 接口(PCIE3.0 x 1)
- Type-C（USB2.0，但默认为调试串口）
- 电源指示灯
