# 接口定义

## 整机接口定义

AIO-3399C 提供了丰富的接口，主要包括：

* 电源接口
* Type-C (OTG)
* 1 x USB3.0
* 2 x USB2.0
* HDMI
* 以太网
* LVDS 屏幕接口
* EDP 屏接口
* TP 触摸接口
* 屏电压跳线接口
* 背光接口
* WIFI 天线
* 蓝牙天线
* 电源按键
* MIC 接口
* 音频输入输出 (linein,lineout)
* 3.5mm 耳机接口
* RTC 电源接口
* 12V 电源接口
* IR 接口
* TF 卡槽
* SIM 卡卡槽
* 扩展按键接口
* I2C
* I2S
* 喇叭接口
* recovery 按键
* EDP 屏接口
* 调试串口
* 工业级串口 (RS485,RS232,2TTL)
* CAN总线
* RTC 电源接入口
* MIPI 屏接口(双 LVDS 复用)
* MINI-PCIE
