## 介绍

**EC-R3588RT** 基于 Rockchip 全新一代旗舰 AIoT 芯片 -- **RK3588S**，采用了 8nm LP 制程；搭载八核（Cortex-A76 x 4 + Cortex-A55 x 4）64位 CPU，主频高达2.4 GHz。  

集成 ARM Mali-G610 MP4 四核 GPU，内置 AI 加速器 NPU，可提供6 Tops 算力，支持主流的深度学习框架；最大支持32 GB 大内存；支持 8K 视频编解码和多种格式的视频输入输出；支持多
种操作系统；可适用于 ARM PC、边缘计算、云服务器、智能NVR 等领域。

![](../../../rk3588_img/EC-R3588RT/Core-3588SG-B.png)  

开发板由核心板 Core-3588J + 底板 MB-G-RK3588S 组成。AIO-3588SG 拥有 RGMII、CAN、USB3.0、I2C、SPI、UART、GPIO、MIPI-DSI 以及 MIPI-CSI 等丰富接口，提供多种的供电方式。可直接应用到各种智能产品中，加速产品落地，详细内容可参考[接口定义](interface_definition.md)。  

![](../../../rk3588_img/EC-R3588RT/AIO-3588SG-B-1.png)  

### AIO-3588SG 标准套装包含以下配件(仅供参考)：
* EC-R3588RT 核心板 x 1
* MB-G-RK3588S 底板 x 1
* 铜管天线 x 3
* Type-C 数据线 x 1

另外可以选购的配件有：

     * 101-M101014-BE45-A1 显示屏 + 转接小板 + 40pin延长线
*    网络
     *   100M/1000M 以太网线缆，及有线路由器
     *   WiFi 路由器
*    输入设备
     *   USB 无线/有线的鼠标/键盘
*    升级固件，调试
     *   Type-C 数据线
     *   串口转 USB 适配器

## 使用须知
### 供电方式

AIO-3588SG 采用电池和TYPE-C两种方式供电：
* 使用电池方式供电，该方式和手机一样，电池没电了需要插上TYPE-C充电,电池规格参考接口使用的Battery章节；
* 使用TYPE-C充电头供电，推荐使用9V3A或12V2A的PD充电头，普通的5V充电器不满足开发板启动要求。


### 显示方式

AIO-3588SG 是没有HDMI显示接口的，有两种方式显示：
* 通过TYPE-C进行投屏或使用TYPE-C接口的DP显示器，此时需要接入电池；
* 使用mipi接口屏幕进行显示，推荐使用官方的mipi屏幕，购买需咨询售后，该方法对电池是否接入不作要求。


### ADB 使用
AIO-3588SG 两种方式接入ADB：
* 使用OTG接入ADB，OTG接口是TYPE-C,也就是说在使用ADB的时候不能使用TYPE-C供电，此时需要接入电池；
* 使用网络ADB，该方法对电池是否接入不作要求。
