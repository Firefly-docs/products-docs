# 登录


AIO-Orin NX 登录方式有两种，一种是通过 DEBUG 进行终端登录，一种是通过 HDMI 登录。

## DEBUG 登录
### Type-C 
拨码开关(**TYPE C-SEL**)先拨到 **DBG**

通过 Type-C 登录方式，用户可以直接参考 Core-1688JD4 维基页面的《[使用 Type-C 串口调试](https://wiki.t-firefly.com/Core-1688JD4/debug.html#shi-yong-type-c-chuan-kou-tiao-shi)》部分，登录账号与密码均为 `nvidia`。

## HDMI 登录

在界面登录的时候，Ubuntu 的会话以及密码均选择 `nvidia` 进行登录。
